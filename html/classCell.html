Detailed Description

Cell is a child of AbstractCell and implements all of its methods, along with overriding the << and = operators. It has two one argument constructors, one of which takes a char that is read in from RunLife.in in the runInput method, and another that creates a new Cell object using a reference to an existing one. Cell acts as a handler for ConwayCell and FredkinCell so that Life can create heterogenous boards.
