Detailed Description

template<class T>
class Life< T >

a template class that creates a board of T cells, which can be either ConwayCells, FredkinCells, or both(Cells). T is read in from RunLife.in in the runInput method.Life passes information to the individual cells so that they can evolve each turn accordingly, and it prints the board along with the live population and generation.
